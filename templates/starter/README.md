## Troubleshooting

- **"Port 5173 already in use"** → another Vite server is running. Stop it, or use a different port: `npm run dev -- --port 3002`.
- **"`command not found: npm`"** → Node isn't installed or isn't on your PATH. Re-install from [nodejs.org](https://nodejs.org) and open a new terminal.
- **"Cannot find module ..."** → dependencies aren't installed or got corrupted. Fix: `rm -rf node_modules package-lock.json && npm install`.
- **Blank white page in the browser** → open the browser DevTools (F12 → Console tab) and paste the first red error into your agent chat.

## Tech notes (for the curious)

- **Vite** serves the dev environment with HMR (changes hot-reload).
- **React Router v7** handles client-side routing (`src/routes/`).
- **shadcn/ui** + **Magic UI** + **Origin UI** components are copied into `src/components/`; feel free to edit them directly.
- **Tailwind v4** — design tokens in `src/index.css` under `@theme`.
